# 首页知识点数量显示为0的问题解决方案

## 问题诊断

经过测试发现，**Neo4j 数据库连接正常，但数据库中没有数据**：
- ✅ Neo4j 连接成功
- ❌ 知识点数量: 0
- ❌ 模块数量: 0

## 原因分析

1. 现有的 `neo4j_init.cypher` 文件是**牙周病学**的数据（标签前缀是 `yzbx_`）
2. 系统需要的是**管理学**的数据（标签前缀是 `glx_`）
3. 数据库没有初始化过管理学的知识图谱数据

## 解决方案

### 1. 创建了管理学数据初始化脚本

文件：`data/init_management_data.cypher`

包含：
- 5个管理学模块（M1-M5）
- 10个章节
- 30个知识点
- 模块-章节-知识点的完整关系
- 知识点前置关系

### 2. 创建了数据导入脚本

文件：`import_data.py`

功能：
- 连接到 Neo4j 数据库
- 读取并执行 Cypher 脚本
- 验证导入的数据
- 显示统计信息

### 3. 导入数据到 Neo4j

**方法一：使用 Python 脚本（推荐）**

```bash
cd "管理学自适应学习系统"
python import_data.py
```

**方法二：直接在 Neo4j Browser 中执行**

1. 打开 Neo4j Browser: http://47.110.83.32:11001
2. 登录账号: neo4j / mima123456
3. 复制 `data/init_management_data.cypher` 的内容
4. 粘贴到查询窗口并执行

**方法三：使用 cypher-shell 命令行工具**

```bash
cypher-shell -a bolt://47.110.83.32:11001 -u neo4j -p mima123456 < data/init_management_data.cypher
```
